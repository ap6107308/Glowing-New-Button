# ✨ Glowing Neon Button

A pure CSS glowing neon button effect built with HTML and CSS — no JavaScript required.

## 🔴 Preview

A centered neon-red button on a black background that lights up with a glowing fill animation on hover.

## 🚀 Features

- Smooth hover transition with skewed fill effect
- Neon red glow using layered `box-shadow`
- Uppercase styled text with letter spacing
- Built with plain HTML & CSS — zero dependencies

## 🛠️ Tech Stack

- HTML5
- CSS3 (transitions, pseudo-elements, box-shadow)
- Google Fonts — [Poppins](https://fonts.google.com/specimen/Poppins)

## 📁 Project Structure

```
├── index.html
└── style.css
```

## ⚙️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/ap6107308/glowing-neon-button.git
   ```
2. Open `index.html` in your browser — that's it!

## 🎨 Customization

You can easily change the button color by replacing all instances of `red` in `style.css` with any color or hex code:

```css
color: #00f0ff;        /* text color */
border: 2px solid #00f0ff;
background: #00f0ff;   /* fill & glow color */
```
